# 🎮 Simon Game

A fun and interactive memory game built using **HTML, CSS, and JavaScript**.
Test your memory skills by repeating the sequence of colors and sounds!

---

## 🚀 Features

* 🎯 Random color sequence generation
* ❌ Game over detection with restart option
* 📈 Increasing difficulty as levels progress
* 🎨 Simple and responsive UI

---

## 🛠️ Tech Stack

* HTML
* CSS
* JavaScript

---

## 🎮 How to Play

1. Press any key to start the game
2. Watch the pattern shown by the game
3. Repeat the pattern by clicking the buttons
4. Each level adds a new step to the sequence
5. Game ends if you click the wrong button
